# Chiru Name Policy

The Chiru project is open source and licensed under the MIT License.

You are free to:

- Use the Chiru source code
- Fork the repository
- Modify and redistribute the code

However, the name **“Chiru”** is reserved to refer to the **official Chiru
language and tooling maintained by the `chiru-lang` organization**.

## Forks and Derivative Works

Forks and derivative works must:

- Use a different name
- Avoid implying they are the official Chiru project
- Avoid using Chiru branding in a way that may cause confusion

This policy exists to protect users and maintain trust in the Chiru language.

## Trademarks

“Chiru” may become a registered trademark in the future.
This document reflects the project’s intent regarding name usage.

Thank you for respecting the Chiru community.
